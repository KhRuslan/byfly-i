# Инструкция по размещению изображений

Для корректной работы сайта необходимо разместить два изображения в папке `public`:

## Необходимые изображения:

1. **`background.jpg`** - Фоновая фотография пляжа
   - Разместить в: `/public/background.jpg`
   - Рекомендуемый размер: минимум 1920x1080px (или больше)
   - Формат: JPG, PNG или WebP
   - Должна содержать: небо, пальмы, океан и пляж

2. **`iphone.png`** - Изображение iPhone с приложением
   - Разместить в: `/public/iphone.png`
   - Рекомендуемый размер: минимум 400x800px (сохранить пропорции iPhone)
   - Формат: PNG с прозрачным фоном (если нужно)
   - Должно содержать: iPhone с экраном приложения ByFly Travel

## Структура папки public:

```
public/
  ├── background.jpg  ← Добавьте сюда
  ├── iphone.png     ← Добавьте сюда
  └── vite.svg
```

После добавления изображений сайт автоматически их подхватит и отобразит.
